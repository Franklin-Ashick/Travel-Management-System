# Travel-Management-System
Travel Management System is a web-based application that helps customers to select countries and book hotels for their travel. 
The project is built using ASP.net and C# programming languages.

The system has a login page where users can log in with their credentials, and a registration page where new users can register. 
For registering, the system uses an OTP verification system, where the user receives an OTP to their registered mobile number for actual verification.

The main feature of the system is the Travel Guide, where users can browse different countries and get detailed information about each country's popular 
tourist destinations, cuisine, and culture. They can also view the available hotels in each country and book them based on their preferences. 
The system allows users to search for hotels based on their budget, location, and other amenities.

The system also includes a dashboard that provides users with real-time information about their bookings, upcoming trips, and other travel-related details.
The dashboard helps users to manage their bookings and make changes to their travel plans, if needed.

Overall, Travel Management System is designed to provide users with a convenient and user-friendly way to plan their travel. The system helps 
users to explore different countries, view available hotels, and book them based on their preferences. The system's use of OTP verification
adds an additional layer of security to the registration process, ensuring that only valid users can access the system. The dashboard provides users with
real-time information about their bookings, making it easy for them to manage their travel plans.
